# Assets Folder

## Required Assets

Place these image files in this folder:

- `icon.png` - App icon (1024x1024 px)
- `splash.png` - Splash screen (1242x2436 px)
- `adaptive-icon.png` - Android adaptive icon (1024x1024 px)
- `favicon.png` - Web favicon (48x48 px)

## Quick Fix

For now, you can use placeholder images or generate them at:
- https://www.appicon.co/
- https://icon.kitchen/

## Temporary Solution

Create a simple colored square as icon.png:
1. Open any image editor
2. Create 1024x1024 image
3. Fill with your brand color (#6200EE)
4. Add text "E-Library"
5. Save as icon.png

Then copy:
- icon.png → splash.png
- icon.png → adaptive-icon.png
- Resize icon.png to 48x48 → favicon.png
